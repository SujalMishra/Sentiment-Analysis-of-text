## React Web Application (ci_project)

The `ci_project` folder contains a React web application. To run the application, follow these steps:

1. Navigate to the `ci_project` folder:
   ```bash
   cd ci_project
2. Install Dependencies :
   ```bash
   npm install
3. Start Frontend :
   ```bash
   npm start
